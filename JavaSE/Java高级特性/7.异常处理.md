# 异常处理

## 学习目标
1. 理解“异常是什么？”以及“异常是干什么的？”
2. 理解异常的两种处理方式
3. 掌握异常的三种类型及区别
4. 掌握如何用 `try-catch-finally` 结构捕获和处理异常
5. 掌握如何用 `throws` 从句声明异常
6. 了解`try-with-resources`语句
